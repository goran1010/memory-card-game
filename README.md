# Memory Card Game

A simple memory card game built with **React** and **Vite**, where players try to click on each Pokémon card only once. The cards shuffle after each click, making it somewhat challenging.

Using [PokeAPI](`https://pokeapi.co) to `fetch` Pokémon cards.

Live preview: https://memory-card-top-project.netlify.app/

## Installation

```bash
# Clone the repository
git clone git@github.com:goran1010/memory-card-game.git

# Navigate to project directory
cd memory-card-game

# Install dependencies
npm install
```

## Usage

```bash
# Development server
npm run dev

# Build for production
npm run build
```

## Playing the game

Open the app in your browser at http://localhost:5173/.

Click on Pokémon cards only once. Each click reshuffles the cards.
If you click a card twice, the game ends and your score resets.

Try to beat your best score.
